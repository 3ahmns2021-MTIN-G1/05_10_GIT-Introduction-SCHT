git init			-> Intitialisierung eines neuen Repositories (Lokal)

git add Dateiname		-> Fügt die Definierte Datei hinzu (Staging Area)
git add *			-> Fügt ALLE Änderungen hinzu (Staging Area)

git commit -m "Eure Nachricht"	-> Erstellt einen Speicherstand (Commit)

git log				-> Zeigt die commit history an
wichtig!!!			-> Die Taste Q-Drücken um aus der Eingabe rauszukommen

git config --global user.email "E-Mail"
git config --global user.name "Name"

git push			-> Ladet meine Commits in die Cloud (GitHub)
git pull			-> ladet meine Commits aus der Cloud ins lokale repos
